## API Reference

#### Get Edurls

```http
  GET /api/edurls/url&des&[apikey]
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `apikey` | `string` | **Required**. Your API key |

#### Get item

```http
  GET /api/people&[apikey]
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `apikey` | `string` | **Required**. Your API key |

#### Get Appointments Data

```http
  GET /api/appos&[apikey]
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `apikey` | `string` | **Required**. Your API key |
